二、应用示例
======
  1.范数计算
  ------
  norm(矩阵，ord = None, axis = None, keepdims = False)
  
  2.求线性方程组的唯一解
  ------
   a = np.array(系数矩阵)
   b = np.array(增广矩阵最后一列（等式右侧解值）)
   x1 = np.linalg.solve(a,b)
   或：
   x2 = np.linalg.inv(a) @ b
   print(x1);print(x2)
   
   3.求超定矩阵的最小二乘解
   -------
   a = np.array(系数矩阵)
   b = np.array(增广矩阵最后一列（等式右侧解值）)
   x = np.linalg.inv(a) @ b
   print(np.round(x,4))
